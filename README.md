# bunsh

An experimental Bun.js shell, because why not?

## Dependencies

- chalk
- cli-table

That's it...

## Setup 

To install dependencies:

```bash
bun install
```

### Running

To build as executable:

```bash
bun build ./index.ts --compile --outfile bunsh
```

Or, to run normally:

```bash
bun run .
```

This project was created using `bun init` in [Bun](https://bun.sh) v1.0.11.
